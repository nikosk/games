# Bit's Grand Adventure

**Existing version:** `code-adventure.html`

## Idea

A grid puzzle adventure where the player writes a short program for Bit, runs it step by step, and repairs a cheerful digital world.

**Player fantasy:** Solve a world problem by telling a small robot exactly what to do.

## Main play

1. Read the goal on a compact grid.
2. Add movement commands to Bit's program.
3. Run or step through the program.
4. See the active command and Bit's result.
5. Fix mistakes and reach the goal.

This should feel like a puzzle, not a lesson or coding worksheet.

## Controls

| Action | Touch | Keyboard/mouse |
|---|---|---|
| Add command | Tap command | Click or shortcut |
| Reorder command | Drag | Drag |
| Remove command | Tap remove | Delete |
| Run, pause, step | Tap controls | Space/S |
| Reset | Tap Reset | R |

## Smallest fun version

- one readable grid
- forward, left, and right commands
- a short program strip
- clear step highlighting
- useful bump or failure feedback
- one satisfying goal puzzle

## If it stays fun

- jump, repeat, and simple conditions
- different tile behaviors
- reusable functions
- themed worlds
- special final puzzles
- a level editor only if it helps us make levels faster

Do not add programming concepts before the previous ones produce good puzzles.

## Look and sound

Friendly pixel-code world, bright readable tiles, an expressive robot, command pulses, soft execution ticks, playful chiptune, and a strong repair celebration.

## Tricky logic worth testing

- command interpreter
- loop limits and function calls
- tile effects
- puzzle validity and solvability where practical
- saved programs and progress

## Ready for the portfolio when

One small puzzle is enjoyable to solve and debug on a tablet, every command's effect is obvious, and failure helps the player make the next change.
