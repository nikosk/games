# Coco's Lost Hat

**Existing version:** `crocodile-game/`

## Idea

Preserve the charming paper-theatre story of Coco searching for a lost hat, then add only the interactions and choices that make the journey more playful and replayable.

**Player fantasy:** Help Coco explore nature, meet friends, solve gentle tactile puzzles, and decorate the recovered hat.

## What already works

The existing game has a clear story, paper-cut visual identity, Greek narration, keyboard support, reduced-motion support, and integrity tests. A rewrite should not throw these strengths away.

## Main play

1. Enter a paper-theatre scene.
2. Tap, drag, or swipe to help Coco.
3. Meet a helper or choose a path.
4. Earn a decoration or reach the next scene.
5. Find the hat and celebrate.

## Controls

| Action | Touch | Keyboard/mouse |
|---|---|---|
| Interact | Tap | Click/Enter |
| Drag or swipe | Touch gesture | Mouse drag/arrows |
| Choose path | Tap choice | Arrow/number key |
| Replay narration | Tap replay | R |
| Decorate hat | Drag item | Mouse drag |

## Smallest fun rewrite

- preserve the current story and Greek narration
- rebuild one short route in Phaser
- retain the layered paper-cut look
- include one tactile puzzle
- include one helper interaction
- finish with hat decoration and celebration

Compare it with the original. Continue only if the rewrite feels better rather than merely newer.

## If it stays fun

- alternate routes
- more nature scenes
- helper-specific interactions
- a decoration collection
- subtitles and another narration language if the owner wants the extra work

No fixed biome, scene, helper, or language count is required.

## Look and sound

Layered paper shapes, watercolour texture, gentle parallax, soft character motion, paper rustles, nature ambience, warm music, synchronized narration, and joyful celebration sounds.

## Tricky logic worth testing

- every route reaches an ending
- puzzle success can be achieved
- narration files and subtitle timing
- save data for routes and decorations
- retained accessibility behavior

## Ready for the portfolio when

The new version preserves the original charm, works smoothly by touch and keyboard, narration is complete for every included language, and the journey feels worth replaying.
