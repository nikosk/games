# Little Chef's Grand Kitchen

**Existing version:** `little-chef-kitchen/`

## Idea

A playful kitchen-building puzzle where ingredients travel through appliances and arrive as finished dishes for animal customers.

**Player fantasy:** Design a magical little food line and watch the kitchen come alive.

## Main play

1. Place a dispenser, belt, and cooking station.
2. Send an ingredient through the line.
3. Process and plate the dish.
4. Serve a waiting customer.
5. Improve the layout so the next order is easier or faster.

The moving kitchen line is the heart of the game.

## Controls

| Action | Touch | Keyboard/mouse |
|---|---|---|
| Choose tool | Tap toolbar | Click toolbar |
| Place station or belt | Tap grid | Click grid |
| Remove or rotate | Context button or repeated tap | Right-click/key |
| Pan | Drag empty area | Mouse drag |
| Pause | Pause button | Escape |

Avoid tiny cells and overlapping gestures. Make placement forgiving.

## Smallest fun version

- one compact kitchen
- dispenser, belt, one processor, plate, and customer
- one recipe from ingredient to finished order
- smooth visible item movement
- a clear success moment
- satisfying appliance and serving sounds

Use the existing game as a reference, but split the oversized scene into a few clear rule modules while rebuilding.

## If it stays fun

- more stations and recipes
- customer patience and varied orders
- chef bots that repeat demonstrated actions
- kitchen layout challenges
- unlockable ingredients
- a relaxed endless service mode

Bots, secret recipes, and endless mode are optional. Add them only after the basic kitchen line is enjoyable.

## Look and sound

Warm illustrated cookbook world, wood and terracotta, expressive animal customers, visible mixer/oven/toaster motion, smooth conveyor travel, sizzles, clatter, dings, and cheerful acoustic music.

## Tricky logic worth testing

- grid occupancy and placement
- belt transfers and junctions
- recipe transformations
- customer order state
- bot replay if added
- save data and any useful old-save import

## Ready for the portfolio when

A player can build and run one complete food line comfortably on tablet, understand why it works or jams, and enjoy serving the finished dish.
