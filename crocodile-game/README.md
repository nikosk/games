# Η Κόκο και το χαμένο καπέλο

Διαδραστική ιστορία για παιδιά 3+ ετών. Tablet-first HTML5 παιχνίδι χωρίς εξωτερικές εξαρτήσεις.

## Τρόπος εκτέλεσης

Ανοίξτε το `index.html` σε οποιονδήποτε σύγχρονο browser (Chrome, Firefox, Safari, Edge). Δεν απαιτείται server, build step ή εγκατάσταση.

```bash
# Απλό άνοιγμα:
open index.html

# Ή με τοπικό server (προτείνεται για αποφυγή CORS με τα audio):
python3 -m http.server 8080
# Μετά ανοίξτε http://localhost:8080
```

> **Σημείωση:** Αν και το παιχνίδι λειτουργεί ανοίγοντας απευθείας το αρχείο, η χρήση τοπικού server συνιστάται για σωστή φόρτωση των αρχείων ήχου.

## Αρχιτεκτονική

```
crocodile-game/
├── index.html        # Κυρίως HTML με όλες τις σκηνές και SVG/CSS γραφικά
```

### Fallback συμπεριφορά

Αν κάποιο αρχείο λείπει, ο ήχος αποτυγχάνει αθόρυβα (silent fail). Το παιχνίδι συνεχίζει κανονικά χωρίς αφήγηση. Δεν εμφανίζονται μηνύματα λάθους στον παίκτη.

## Χαρακτηριστικά

- **Picture-book αισθητική:** Στυλ «paper-cut» με επίπεδα βάθους, οργανικές φόρμες, ζεστή χρωματική παλέτα
- **Γραφικά:** Όλα τα γραφικά είναι SVG/CSS — δεν χρησιμοποιείται canvas
- **Προσβασιμότητα:** Πλήρης υποστήριξη αφής, ποντικιού και πληκτρολογίου, ορατά focus states, aria-labels
- **Responsive:** Landscape και portrait, υποστήριξη safe-area (notch), μεγάλοι στόχοι αφής
- **Ασφάλεια:** Χωρίς διαφημίσεις, χωρίς σκορ/χρονόμετρο/ήττα, χωρίς εξωτερικά dependencies
- **prefers-reduced-motion:** Σέβεται τις ρυθμίσεις του χρήστη για μειωμένη κίνηση
- **Πρόληψη διπλών αλληλεπιδράσεων:** Debounce 600ms σε όλα τα κουμπιά
- **Πρόληψη scroll/zoom:** Απενεργοποιημένο το pinch-zoom και το double-tap zoom στην περιοχή του παιχνιδιού
- **Fullscreen:** Κουμπί πλήρους οθόνης
- **Mute/ replay:** Κουμπιά σίγασης και επανάληψης αφήγησης
- **Πρώτο πάτημα:** Το audio ξεκλειδώνει με το πρώτο άγγιγμα του χρήστη

## Έλεγχοι

Εκτελέστε τοπικούς ελέγχους:

```bash
# Έλεγχος σύνταξης HTML
npx html-validate index.html 2>/dev/null || echo "Το html-validate δεν είναι διαθέσιμο"

# Έλεγχος σύνταξης CSS
npx stylelint styles.css 2>/dev/null || echo "Το stylelint δεν είναι διαθέσιμο"

# Όλοι οι έλεγχοι χωρίς εγκατάσταση dependencies
npm run check

# Ή μόνο έλεγχος σύνταξης JavaScript
node --check game.js && echo "JS syntax OK"
```
