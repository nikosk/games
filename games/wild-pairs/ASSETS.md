# Asset and sound note

The 18 animal card portraits under `assets/images/` were generated with `bytedance-seed/seedream-4.5` through OpenRouter. Each final prompt is preserved under `assets/source/`. The fox portrait was used as the visual reference for the other animals so the set keeps a consistent children's watercolour field-journal style.

The meadow, journal paper, card frames, interface art, and effects are drawn at runtime with Phaser Graphics. Flip, mismatch, match, and completion cues are original tones synthesized at runtime with the Web Audio API; no sourced recordings or third-party assets are used.
